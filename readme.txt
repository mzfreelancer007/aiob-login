== Description ==

This is a easy plugin to create your own small form for login. add this plugin and user shortcode [aiob-login]

A few notes about the sections above:

*   Just put shortcode [aiob-login] on page and you'll get your login page done.
*   Tested code, also usable to any kind of theme.
